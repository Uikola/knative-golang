## Начало Работы

Чтобы запустить приложение следуйте следующим шагам.

### Запуск

1. Клонируйте репозиторий
   ```sh
   git clone https://github.com/Uikola/knative-golang.git
   ```

2. Запустите go mod tidy:
   ```sh
   go mod tidy
   ```

3. Запустите сервер
   ```sh
   go run main.go
   ```

4. Приложение готово к использованию!
